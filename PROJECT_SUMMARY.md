# Educational Institute Management System

## Project Overview

Educational Institute Management System is a web application developed using React JS and Tailwind CSS.

The system helps educational institutes manage enquiries, admissions, reports, and dashboard statistics from a single interface.

---

## Technology Stack

### Frontend

* React JS
* React Router DOM
* Tailwind CSS
* JavaScript (ES6+)

### Planned Backend

* Node.js
* Express.js

### Planned Database

* MySQL
